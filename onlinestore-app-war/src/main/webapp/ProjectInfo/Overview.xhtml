Facelet Title Overview

This is an online store application based on Java EE that I created as a learning project. The source code and UML are available here, and list of the features is here. I have tried to make it realistic. This application should be usable by a real business if it is properly deployed, customized according to the business rules, and decorated with more attractive front-end design.

You can log in with several user roles, each one with different privileges
- Customer: able to browse catalog and place orders
- Employee:
 - business manager: able to change the catalog of products sold and their prices
 - accountant: able to view list of orders and payment information, e.g. credit cards.
 - warehouse: able to view list of orders and shipping address, update product availability, mark order as shipped
 - admin: able to view list of accounts, change user roles and privileges (create custom accounts). Only the administartor can create emploeyee accounts.
