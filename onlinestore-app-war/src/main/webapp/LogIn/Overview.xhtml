Facelet Title Overview

This is an online store for selling books and CD's. I developed it to practice with some of the common Java Enterprise technologies and tools. The source code, UML, business rules, and documentation are available here.

The application has not only customer user roles (for placing orders) but also employee user roles (for managing products and processing orders).

I hope that you check it out and let me know what you think. You can:

  • the store and place orders without logging in OR
  • a customer account and log in OR
  • as one of the pre-defined DEMO accounts, some of which have employee user roles (password is simply p)
    • username = e (Everyone, has all the user roles described below) <= Recommended
    • username = c (Customer user role, can place orders)
    • username = m (Manager user role, can manage products and authorize order returns)
    • username = w (Warehouse worker user role, can process order shipments)
    • username = a (Accountant user role, can process order payments)

This applicaiton is currently being hosted on a Glassfish 3.1 server and MySQL database on my home computer.
