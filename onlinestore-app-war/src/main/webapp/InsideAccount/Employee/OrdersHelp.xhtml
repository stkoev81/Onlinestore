How order processing works

The diagram below describes the order processing sequence. Tips:

  • Accountants: Before you start processing a payment, update order status to PAYMENT_PROCESSING. Similarly, before you start processing a refund, update status to REFUND_PROCESSING. This will prevent other accountants from working on the same payment.
  • Warehouse workers: Before you start processing a shipment, update order status to SHIPMENT_PROCESSING. This will prevent other warehouse workers from working on the same shipment.
